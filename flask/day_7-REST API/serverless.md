# Serverless Computing

## What?

  - No servers to manage on your own 

## Why?

  - Less maintenance, Less planning
  - Huge cost svaing

## How?
  - Lambda
  - API Gateway
  - and much more



====

EC2 instances - Managed by You (Billed Hourly)
Managed Services - AWS Managed (RDS etc.)
Serverless Offerings - Lambda, API Gateway.

EC2 vs Lambda:

EC2 - up all the time
Lambda - on demand
